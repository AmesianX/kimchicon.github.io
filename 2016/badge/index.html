김치콘 뱃지엔 총 3개의 하드웨어 해킹 문제가 숨겨져 있습니다. 김치콘 행사 기간동안 Binary Reversing & Code Auditing을 통해 숨겨진 문제를 찾고 해결해 보세요. 심사위원의 평가에 따라 우승자에게는 "Play Station 4"를 상품으로 드립니다.

김치콘 뱃지는 참가자들이 펌웨어를 자유롭게 수정 가능하도록 제작되어 있습니다. 김치콘 첫째 날 저녁에는 김치콘 뱃지를 개조하는 해커톤이 진행되며, 김치콘 기간 동안 가장 기발하고 멋진 작품을 만들어주신 분께 "건담 프라모델"을 상품으로 드립니다.

 

* 하드웨어 해커톤 참가자는 당일 접수를 받습니다.

* 해커톤 장소는 Kaist System Security Lab이며, 시간은 저녁 7시~늦은 밤까지입니다.

* 해커톤에 필요한 장비/부품들은 제공/판매되며, 직접 준비해 오시는 것도 환영합니다.

김치콘 뱃지내에 포함되어 있는 Flappy Bird 게임의 최고득점을 획득하신 분께 "드론"을 상품으로 드립니다. 승리하고 싶다면, Flappy Bird를 미리 연습하고 오세요!

 

* 본 이벤트는 예선과 본선으로 나뉘어 진행됩니다.

* Hacking을 통해 고득점을 획득하는 것은 본 이벤트에선 무효입니다.

김치콘 행사 폐회식에서 참석자들의 뱃지를 대상으로 random 추첨을 진행합니다.

해당 뱃지엔 특정 신호가 전달되며, 당첨된 분들께는 "드론"을 상품으로 드립니다.

폐회식 때까지 뱃지의 배터리가 닳지 않도록 주의해 주세요. :)

 

* 김치콘 뱃지는 USB cable을 통해 충전할 수 있습니다.



  MCU : Atmega32U4 (8bit, 16Mhz)

  Flash : 32KB

  Memory : SRAM 2.5KB

  Display : ST7735 1.8'' TFT-LCD

  WIFI : ESP8266

  Battery : Li-Po 1500mAh

  Operating Voltage : 2.7~5.5V

  Firmware Core : Arduino Leonardo

  주요기능 : 공지사항, 스케쥴, 게임, 갤러리, 찬반투표 등
