Abstract

페이스북은 특유의 간편한 UI와 유용한 서비스로 인해 전 세계적으로 가장 널리 사용되고 있는 소셜 네트워크 서비스이다. 페이스북은 사용자간 친구 관계를 지속적으로 증진하기 위하여 다양한 사용자 검색 기능을 제공한다. 우리는 이러한 검색 옵션 중, 사용자의 전화번호를 이용하는 검색 방법을 이용하고 페이스북 사용자의 개인 정보(사용자 이름, 사용자 사진, 사용자 전화번호, 거주 지역, 직장, 학력, 가족 관계, 친구, 이벤트 날짜 등)를 수집할 수 있는 새로운 열거 공격 (enumeration attack) 방법을 소개한다. 다양한 실험 방법을 통하여 실제 페이스북에서 효과적인 열거 공격 조건을 찾아내고, 공격에 대한 자동화가 가능함을 보여준다. 제안 공격의 효과를 증명하기 위하여, 실제 수 만명의 페이스북 사용자들의 개인 정보를 단 시간에 수집할 수 있다는 것을 실험을 통하여 증명한다. 페이스북에서 사용하고 있는 다양한 안티 크롤링 기법을 회피하기 위한 방법을 소개하고, 이러한 공격을 막기 위한 실질적인 방법들을 추천한다.

DELTA: SDN SECURITY EVALUATION FRAMEWORK

이승수

KAIST

Profile

- Network and System Security (NSS) Laboratory, KAIST
- Research interests : network security, software-defined networking security

Abstract

오늘날 소프트웨어 정의 네트워킹(Software-Defined Networking, SDN)이 미래 인터넷 아키텍처를 구성할 주요한 패러다임으로써 받아들여지고 있다. 산업계에서는 국내/외 데이터센터나 엔터프라이즈 네트워크의 관리자들이 적극적으로 SDN을 도입하려 하고 있으며, 학계에서는 SDN을 이용하여 새로운 네트워크 시스템을 제안하는 연구가 활발히 제안되고 있다. 더불어, SDN의 보안 취약점에 관한 여러 연구들 또한 제시가 되었는데, 특히, 중앙 집중화된 SDN 컨트롤러를 공격함으로 공격자가 전체 네트워크 성능/기능에 영향을 미칠 수 있음이 증명되었다. 이는 곧 SDN이 본격적으로 도입되기 전에 보안성에 관해 철저한 검증이 이루어져야 한다는 것을 시사한다. 본 발표에서는 이러한 SDN의 전반적인 보안성을 점검하기 위한 침투 테스트 도구인 DELTA를 소개하고자 한다.

SMB 너로 정했다!

이지훈, 최진영

Blackperl Security

Profile

- Security Researcher of Blackperl Security
- Research interests : System security

Abstract

Microsoft Windows SMB 원데이 취약점을 이용한 랜섬웨어 WannaCry가 전세계를 위협하고 있다. 기존의 랜섬웨어가 주로 Drive-by-Download 취약점 등 사용자 행위 기반으로 감염되는 것과 다르게, WannaCry는 사용자의 행위와 상관없이 OS 내의 SMB 취약점을 이용해 감염된다.

SMB(Server Message Block) 란 윈도우즈에서 파일이나 디렉터리 및 주변장치들을 공유하는데 사용되는 메시지형식이다. 많은 네트워크 제품들에서 사용되며, 서로 다른 운영체제에서 파일을 공유할때도 사용된다. Windows OS에 기본적으로 탑재되어 있는 기능 중 하나이기 때문에 해당 OS를 사용하는 다수의 기기(PC, ATM, POS, Server등)가 랜섬웨어 감염 대상이 된다.

해당 발표에서는 SMB 취약점을 분석 및 취약점을 찾기 위한 과정을 소개한다.

SMB 프로토콜 구조 분석 및 기존 취약점을 이해하는 시간을 가진다. 또한 새로운 SMB 취약점을 찾기 위해 시도한 방법들을 이야기하고, 취약점을 찾기 위해 더 시도해 볼 수 있는 방법들을 제안한다.
